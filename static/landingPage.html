PyFlash Reminders

An SMS-based reminder application, which utilizes a python microframework called Flask to build an app that will be able to receive and send SMS. Users can create an account through this website or the android/iOS app. With an account, a reminder can be sent to the number provided and it will remind you on time. Never miss or forget anything ever again!
